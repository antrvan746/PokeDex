# What are we building?
We're going to build a Pokedex in a command-line REPL. We'll use the Pok√©API to power all of the data we'll need. A Pokedex is just a make-believe device that lets us look up information about Pokemon - things like their name, type, and stats.

# The learning goals

+ Learn how to parse JSON in Go
+ Practice making HTTP requests in Go
+ Learn how to build a CLI tool that makes interacting with a back-end server easier
+ Get hands-on practice with local Go development and tooling
+ Learn about caching and how to use it to improve performance
